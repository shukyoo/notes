## 系统
centos 7

## 步骤
```
vim /usr/lib/systemd/system/docker.service

// [Service]下添加
EnvironmentFile=-/etc/sysconfig/docker

// ExecStart=/usr/bin/dockerd -H fd:// --containerd=/run/containerd/containerd.sock $OPTIONS 这一行后面加 $OPTIONS

vim /etc/sysconfig/docker

// 添加以下
#选择网桥
#DOCKER_OPTS="-b=br0"
#指定DNS
OPTIONS="--dns 114.114.114.114"

// 重启
systemctl daemon-reload
systemctl restart docker 

// 接下来查看docker进程，发现后面已经追加了dns，这样容器可以访问外网了
```
